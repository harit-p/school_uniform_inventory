# Optional fonts for PDF invoices

To render **Gujarati** text on bills (શ્રી હરિ વસ્ત્ર ભંડાર, etc.), download:

**Noto Sans Gujarati** (Regular) from [Google Fonts – Noto Sans Gujarati](https://fonts.google.com/noto/specimen/Noto+Sans+Gujarati) and save as:

`server/fonts/NotoSansGujarati-Regular.ttf`

If this file is missing, the PDF uses English labels for those lines.
